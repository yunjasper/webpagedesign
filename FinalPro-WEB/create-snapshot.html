How to capture a snapshot of a webpage

Here is one possible way to do it:

  1. Open your webpage in a browser.
  2. Click the Start button.
  3. Enter "snipping" in the Search programs and files box.
  4. You should see an entry under Programs called Snipping Tool.
  5. Start the Snipping Tool by clicking the menu entry.
  6. Use your mouse to draw a box around the area you want to capture.
  7. When you release the mouse button, the captured image will appear in the Snipping Tool window.
  8. Save it as a jpg or png file
  9. Use it!
Now click on the Final Project window to continue!
